# Trace visualizer
This folders contains all the Python scripts we have developed to re-produce HPCC plots:
- `/data` : contaians input files for generating plots and generated plot figures;
- `/scripts` : contains Python scripts (see `/scripts` for more details).

Since input files for plot generation were considerably heavy, we put `/data/input` in `.gitignore` and uploaded input files at [this](https://drive.google.com/drive/folders/1oKYEBdR2jjJCV2JnJJ73cf9ubTwE_ZN2?usp=drive_link) Google Drive folder.
